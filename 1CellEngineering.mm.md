# 1Cell Engineering

- AI/ML Team
  - Data Scientist
    - Build predictive and analytical models
    - Discover patterns and insights in data
    - Feature engineering and data exploration
    - Experimentation and hypothesis testing

  - Lead ML Engineer
    - Optimize model performance and scalability
    - Design model architectures
    - Train and fine-tune models
    - Convert research models into production-ready systems

  - MLOps Engineer
    - Run models in production environments
    - Operate AI systems at scale
    - Build CI/CD pipelines for ML
    - Monitor model performance, drift, and reliability
    - Manage model versioning and rollback

  - AI Engineer
    - Build AI-powered products and applications
    - Implement guardrails and safety systems
    - AI explainability and interpretability layers
    - Build RAG, agents, and orchestration systems
    - Integrate AI into real business workflows

- Software Engineering Team
  - Solution Architect
    - Defines overall system architecture
    - Aligns business requirements with technical design
    - Designs cross-team integration patterns
    - Reviews high-level designs and APIs
    - Ensures scalability, security, and compliance
    - Guides technology and platform choices
  - Platform API Team
    - Owns core backend services
    - Designs domain-driven APIs
    - Manages data models and persistence
    - Implements authentication and authorization
    - Ensures performance, reliability, and scalability
  - Business API / GraphQL Team
    - Builds aggregation and orchestration layer
    - Exposes client-optimized APIs
    - Implements business rules & workflows
    - Manages schema evolution and versioning
    - Decouples frontend and backend systems
  - Full Stack Engineers
    - Deliver end-to-end product features
    - Consume Business APIs / GraphQL
    - Implement frontend logic and backend extensions
    - Handle rapid iterations and bug fixes
    - Collaborate closely with UX and Product
  - Mobile Engineers
    - Develop iOS and Android applications
    - Integrate APIs and device capabilities
    - Optimize performance and offline behavior
    - Manage app builds and store releases
    - Ensure secure handling of user data

- UI/UX Team
  - Product Designer (Lead)
    - Owns end-to-end design
    - Translates product requirements into UX flows
    - Creates wireframes and high-fidelity designs
    - Works closely with PM & Engineering
    - Maintains overall UX quality and consistency
  - UI/UX Designer
    - Designs visual interfaces
    - Builds and maintains design system
    - Creates reusable components
    - Ensures brand consistency
    - Delivers pixel-perfect screens

- Database Team
  - Database Administrator (DBA)
    - Installation & Upgrades
    - Backups & Recovery
    - Performance Tuning
    - Monitoring & Alerting
    - Security & Access Control
    - High Availability
    - Archiving & Retention
    - Capacity Planning
  - Database Developer
    - Schema Design
    - Data Modeling
    - SQL / NoSQL Queries
    - Stored Procedures
    - Indexing Strategy
    - Query Optimization
    - Data Integrity
    - Application Integration

- DevOps Team
  - DevOps / Cloud Engineer
    - AWS infra
    - CI/CD
    - deployments
    - automation
    - IAM & Secret Management
    - Basic Security
  - SRE / Reliability Engineer
    - Monitoring
    - uptime
    - incidents
    - scaling
    - Compliance & Auditing

- SQA Team
  - QA Lead / Test Manager
    - Owns overall quality strategy
    - Defines test processes & standards
    - Plans release testing and sign-off
    - Coordinates with Product & Engineering
    - Tracks quality metrics and risks
  - Manual QA Engineer
    - Writes and executes test cases
    - Performs functional, regression testing
    - Validates UI/UX and user workflows
    - Reproduces and documents bugs
    - Verifies fixes across builds
  - Automation QA Engineer
    - Designs and maintains test automation
    - Builds CI-integrated test suites
    - Covers regression and smoke tests
    - Improves test reliability & speed
    - Reduces manual testing effort
  - Performance / Security QA (Optional)
    - Executes load and stress testing
    - Identifies performance bottlenecks
    - Validates basic security scenarios
    - Supports compliance requirements

- Program Team
  - Product Owners
  - Project/ Program

- IT Team
  - IT Support Lead / IT Manager
    - Owns office IT infrastructure
    - Defines IT policies and standards
    - Manages vendors and licenses
    - Plans hardware procurement
    - Handles escalations and security incidents

  - Desktop / Helpdesk Support Engineer
    - Sets up laptops and workstations
    - Installs OS, software, and tools
    - Troubleshoots hardware issues
    - Provides day-to-day user support
    - Manages asset inventory

  - Network / Systems Administrator
    - Sets up routers, switches, and Wi-Fi
    - Manages LAN, VPN, and firewalls
    - Monitors network uptime and performance
    - Configures servers and shared services
    - Handles backups and access controls

- Product Support & Training
  - Head of Product Support / Customer Success
    - Owns customer support strategy
    - Defines SLAs and escalation paths
    - Aligns support with product roadmap
    - Tracks customer satisfaction metrics
    - Coordinates with Engineering and Product

  - Product Support Engineer (L1/L2)
    - Handles customer-reported issues
    - Troubleshoots product and configuration problems
    - Reproduces bugs and logs tickets
    - Coordinates fixes with Engineering
    - Provides workaround solutions

  - Product Specialist / SME
    - Deep product and domain expertise
    - Supports complex customer use cases
    - Assists with pre-sales and onboarding
    - Validates product configurations
    - Acts as bridge between customers and Product team
